# ppengo

pptr + Express + node.js + mongodb

## Quick Start
1. Clone the repository
```bash
$ git clone <repository>
$ cd ppengo
```

2. build and start ppengo
```bash
$ docker-compose build
$ docker-compose up -d
```

3. build and start wgeteer 
```bash
$ cd wgeteer
$ docker-compose up -d
```

4. Open localhost:3000/ppengo in the web browser
